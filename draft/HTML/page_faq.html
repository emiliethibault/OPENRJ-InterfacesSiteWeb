Frequently Asked Questions

Cette FAQ a été réalisée à partir des questions fréquemment posées. Si jamais certaines informations dont vous pourriez avoir besoin sont manquantes, n'hésitez pas à nous contacter. Nous ajouterons votre question dans cette section.

La plateforme OPENRJ met à disposition des données de consommations énergétiques de bâtiments tertiaires. Les données comprennent : les mesures de consommations, les méta-données de ces données (unités, ...). Ces données sont liés à des sites, des bâtiments pour lesquels vous pouvez obtenir des informations (année de construction, ...)

Pour télécharger les données, deux chois s'offrent à vous :

  • Vous êtes développeur, rendez-vous sur la page de documentation des API.
  • Vous êtes un particulier et vous souhaitez simplement télécharger les données. Celles ci sont acessibles dans la page Rechercher des données. Sélectionner les données qui vous intéresse et cliquer sur le bouton pour les télécharger au format csv (excel).

Dans la page Rechercher des données, vous pouvez noter les jeu de données en cliquant sur les étoiles. Si vous souhaitez nous faire remonter l'information directement, contactez-nous via le formulaire de contact en sélectionnant "Validation de jeux de données" comme objet.

La plateforme OPENRJ est capable d'accueillir les données de n'importe quel système de mesure. La preuve, nous remontons actuellement les données de 3 systèmes complétement différents.

  • GTB/GTC
  • Deltadore
  • Solution QUALISTEO
  • ...

Il est nécessaire de mesurer les consommations énergétiques de vos bâtiments si vous souhaitez participer au projet. N'importe quel système fonctionne.

Si vous le souhaitez, QUALISTEO, entreprise de la région PACA, peut auditer vos bâtiments et vous proposez des solutions techniques de mesures, faciles à mettre en place et non intrusives.

Vous trouverez toute la document nécessaire au transfert de vos données vers la plateforme OPENRJ dans la page Documentation - API.

Si certains points vous pose des difficultés, n'hésitez pas à nous contacter.
